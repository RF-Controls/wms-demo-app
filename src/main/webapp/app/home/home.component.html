RF Controls Warehouse Management Demo

A small application for relating RFID tags to real world items.

Identify

Uniquely identify multiple passive RFID tags with an exceptionally long read range.

Locate

Accurately ascertain the present location of all passive RFID tags in three dimensions (x, y, z).

Track

Track the movement of passive RFID tags concurrently in real-time.

Getting Started

Create a region in RFC OS
Be sure you have created and configured a Region in RFC OS and it is up and running.
Create an Association Zone
This is done inside this application so you can easily associate existing EPC codes to inventory Items
Identify an Item
Place the RFID tagged item into the association zone near an antenna
Associate Item
Create an Item in the system and select the EPC code identified within the association zone.
Track Item
Item locations can now be searched on the Item Location page.
